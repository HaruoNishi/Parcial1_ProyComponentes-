PokéApi

En este proyecto se implementa esta API pública gracias al uso de Ajax y JavaScript. la PokéApi nos proporciona ciento de información de diferentes ítems relacionados al mundo Pokémon, como lo son:

  • Pokémons (incluyendo sus distintas formas)
  • Movimientos
  • Habilidades
  • Tipos
  • versiones de juego
  • objetos
